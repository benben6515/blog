決擇之日

發表於 2021-06-03 分類於 程式筆記 Disqus：

時間很快到了第 8 周，也是 MTR05 的決擇之日。

看到訊息的時候，我還是猶豫了…

猶豫

該選什麼方案呢？其實在一開始就想好要選 A 方案了，但是還是猶豫了，畢竟兩個方案很可能會差到 10000 塊 ( 很快算了一下，差不多月薪 45000 、不含年終，學費: 64800 ) ，想想 10000 也可以做不少事，像是買個好的雙營幕跟好的鍵盤都還有找呢 ( ㄟ？不是 )

我是想說這 10000 塊，對我來說很重要。

It’s mean something to me.

而且會分這兩個方案，也是因為要幫我們下定決心吧，這部分對於我 ( 或是比較自律的人 ) 好像就還好，因為選 A 或 B 課程內容都一樣，不會有差別

分析

之前也在 Gather.town 小樹屋跟其他同學還有助教聊過，也有學長是選 B 方案的，他是這麼分析的 :

你看選 B 方案的話，可以不用寫每日進度跟寫畢業心得 XD

還有人說最佳解是選 A 然後，兩個月找不到工作，就免學費 (？？？)，真的笑死

決定

想了很多，我在想 : 會不會這個決定，會影響之後工程師的一生 ( 才沒這麼嚴重啦 )，想了一晚，還是決定選 A 方案，如果學費越多就代表我的薪水是越多的，這樣想著想著好像就很有動力XD

最後想了這麼多，還是回到原點，但不一樣，這是一種見山又是山的感覺 ( 在那邊自以為 )

總之，我選 A plan
