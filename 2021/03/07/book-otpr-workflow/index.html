OTPR 敏捷工作法 | OTPR Workflow

發表於 2021-03-07 分類於 閱讀筆記 Disqus：

ISBN:978-957-13-8274-6

- 游舒帆 (Gipi)

拿回績效主導權，讓工作更快、更好、更有價值。

推薦序

  • 對你足夠重要的是，得用盡方法讓它發生。
  • 讓人看不到車尾燈的秘密
  • 如果不主動管理時間，將會被取代
  • 用OTPR工作法，抓回人生主導權

前言 : 讓工作做得更快、更好、更有價值的秘密。

  • 內部歸因的人會從自己身上找答案；外部歸因的人則會認為一切都是別人的問題。
  • 一套工作法，很高效，而且調整速度很快。

為何多數人無法為自己的績效負責

  • 價值誤區 : 以為做事就有價值，錯把苦勞當功勞
  • 目標誤區 : 錯把手段當目的，與老闆目標出現分歧
  • 分工誤區 : 將自己的績效寄託於老闆、同事身上
  • 高能低效 : 能者多勞，無止境的低價循環

拿回自己績效主導權的OTPR敏捷工作法

  • O : OKR, Objective and Key Results
  • T : Time Management
  • P : Project Management
  • R : Retrospective
  • 常見問題 : 排計畫好難、變化性太大

OTPR的具體示範

  • 先擬定周計畫，找到工作的優先順序
  • 工作分類 : 緊急任務、插單工作、規劃或執行缺失、自發性任務變更
  • 提早規劃協同性工作 : 了解他人的原則，同時讓對方理解你的原則
  • 有效持續運用OTPR工作法

    Plans are nothing, planing is everaything. - 美國名將艾森豪

O - OKR ，設定目標是高績效源頭

  • 目標設定的基本概念 : SMART、OKR
  • 釐清與設定個人目標 : 將工作目標與個人目標結合
  • 盤點非相關工作的個人目標 : 八大類目標 X 想追求的、想改善的

    Specific Measurable Attainable Relerant Time-bound
    傳統 KPI (Key Results Indictors) 的盲點

    • 基層員工不知道KPI背後的目標
    • 基層與高層看到的現況不一致

      八類目標 : 成就、紀律、生理、心理、財務、能力、人際、興趣

T - Time Management，用時間管理那些難事

  • 實際盤點工作之外的目標 : 目標產生衝突如何取捨?
  • 生活是一個整體 : 先保障基本需求，再找出目標脈絡性
  • 為目標設定行動方案 : 盤點資源、進行排序
  • 重要性的排序是一大學問 : 利用二維矩陣，建立優先順序

    目標的脈絡性

    • 目標彼此之間並非獨立的，而是依存的關係
    • 時間是最主要的限制資源

P - Project Management，安排與管理每周的行程

  • 安排計畫前的五個重要原則 : 定義交付產出、估算工作量、六小時原則、不開會原則、留時間給自己
  • 將工作排進行事曆 : 主動安排行程拿回主導權
  • 對行程進行模擬 : 提早選擇，排除不確定性
  • 執行計畫 : 按表操課與應變管理

    常用行事曆分類 :

    1. 工作行事曆
    2. 協同性工作
    3. 個人工作目標
    4. 變更行事曆
    5. Re-plan
    6. 復盤行事曆

      變更管理 :

    7. 變更一定會發生
    8. 減少變更是我們的責任
    9. 有價值時，歡迎變更
    10. 減少變更後得衝擊

R - Retrospective，回顧與復盤

  • 日回顧 : 對過去的事盤點、檢討與自我提問
  • 周回顧 : 準備時間表，以週為單位的事
  • OTPR的下一步 : 延長周期與休耕期

    可以今天就修正的事，為什麼要等到周末呢?
    自我提升的好時機 :

    1. 計畫與實際之間有明顯落差時
    2. 行為與思維不一致時
    3. 與他人無法取得共識時

      讓休耕期成為你生活中的一種固定儀式，提醒自己得停下來想想

漸進式的學習，OTPR的團隊應用

  • 常見方式 : 個別使用、從P與R切入、找學伴
  • OTPR使用經驗分享 :
    • 善用Google行事曆
    • 回顧最初的OKR，確認最想完成的事
    • 使用週迭代來做時間管理
    • 有意識改變自己的行為，慢一點沒關係
  • OTPR的團隊運用四步驟 :
    1. 起手式 : 週計畫 + 日回顧
    2. 第二步驟 : 週回顧
    3. 第三步驟 : 打通向上與橫向關係
    4. 第四步驟 : 完整落實OTPR
