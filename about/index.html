About this site

I’m Benben, a front-end developer from Taiwan and love both React and Vue framework.

I’m also a tech writer, math tutor, learning junkie.

This is Benben’s little universe which is a blog about web tech, English learning, book reading documentary.

本站說明

嗨，你好！

這裡是 Benben 的小宇宙，為東西比較雜一點，基本上會有一些 Web 技術文章、閱讀筆記、英文學習，可能會有一些全英文的筆記，主要是練習用，所以可能會有些英文的錯誤或是待整理的，造成不便先說聲 Sorry ，有任何錯誤也迎歡隨時跟我說，我會盡快更新的！

關於我 | About me

嗨！我是 Benben，是一位：

  1. 前端開發者（Front-End Developer）
  2. 技術文章寫手（Tech Writer）
  3. 數學家教（Math tutor）
  4. 學習成癮者（Learning junkie）

喜歡理性的冷知識、感性的人事物。

平時歡喜看書、寫寫文章筆記，當然還有寫扣。

不太喝咖啡，但喜歡喝無糖紅茶（最愛約翰紅茶 - 茶中香檳、鶴茶樓 - 綺夢紅茶）

以下看不懂的名詞，跳過就可以了

主要沉浸在 React 跟 Vue 生態系（Angular 表示：？），最近在學習 TypeScript, Node.js/Express，還有英文，因為英文沒有學完的一天。

會想學習這些技術，是因為目前市場上新的主流都用這些技術，加上時間有限，應該投資在 CP 值高一點的技能。

雖然是 1995 年生，但非常老派的（？）喜歡使用 Vim （VScode plugin, NeoVim） 跟 嘸蝦米輸入法，這兩個技能再加上 英文 就算是學習程式的「九陽神功」，發現不少資深的前輩也是這個技能組，學什麼都快、打字快、看文件寫文件也快，感謝網路讓我們有這些資源，我們活在最好的世代。

學程式的起蒙是 Lidemy ，是一個教你用自己能力去學習程式的地方，不只給你魚又教你釣魚，也很開心在這裡遇到的每一個人，當然還有我最感謝的恩師 胡立 | Huli。

因為有你們，我才懂得：

也許一個人可以走得很快，
但一群人可以走得遠，
很遠、
很遠。

也很感謝你的閱讀。

推薦的課程清單 | Online course list

  • 前端課程心得感想

因為蠻多朋友跟讀者問如何入門的，就簡單整理一下，供讀者參考囉，筆者個人是真的買了很多很多線上課程，除了 Lidemy 外，Udemy, Hahow, HiSkio 我全都有買，全部加一加應該快 100 堂了，當然也收藏了很多文章、資源，入門最推的還是 Udemy ，沒什麼特別原因 CP 值高，一堂也才 300 多，相信是任何人都負擔的起，其實台灣也 不少課從 Udemy 上搬運來賣的，筆者就不多說了（汗）。

  • Lidemy 程式導師實驗計畫第五期

暫時停止招生了，2021/10/01 最後一屆畢業，非常開心我是其中之一。那時候參加有 A、B 方案，A 是保証班，轉職完後第一個報到 Offer 的年薪 12%；B 是課程方案，不管有沒有 Offer 都是一口價 55000 。兩者都包含半年的課程、直播、文章內容，最後我選擇了 A 方案然後付了 7 萬出頭的費用，問我值得嗎？讀者認為呢？然當我必須說 這並不適合每個人，但果如都跟補習班、線上課程廣告不實，有點不負責任，身為筆者一定要對讀者負責任。

  • Zero To Mastery/

ZTM 是 Udemy 上非常優質的學習資源，推薦先收藏等特價買，因為 Udemy 上常有特價，當然 ZTM 也有自己的網站：https://zerotomastery.io/，有訂閱制、終身制等等，對了買之前先網路上找個 coupon 真的會差很多，筆者個人是買終身制的供參考，至於為什麼還要再買這個呢？我不是已經上完 Lidemy 了嗎？等有興趣的讀者去做點功課吧～
