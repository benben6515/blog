關於我

嗨！我是 Benben，是一位前端開發者（Front End Developer）。

喜歡理性的冷知識、感性的人事物。

平時歡喜看書、慢跑，當然還有寫扣。

主要沉浸在 React 跟 Vue 生態系（Angular 表示：？），最近在學習 TypeScript, Node.js/Express，當然還有英文。

會想學習這些技術，是因為目前市場上新的主流都用這些技術，加上時間有限，應該投資在 CP 值高一點的技能。

雖然是 8 年級生，但非常老派的（？）喜歡使用 Vim （VScode plugin） 跟 嘸蝦米，這兩個技能再加上 英文 就算是學習程式的九陽神功，不少資深的前輩也是這個技能組，學什麼都快、打字快、看文件寫文件也快，感謝網路讓我們有這些資源，我們活在最好的世代。

學程式的起蒙是 Lidemy ，是一個教你用自己能力去學習的地方，也很開心在這裡遇到的每一個人。

因為有你們，我才懂得：

一個人可以走得很快，
但一群人可以走得遠，
很遠、
很遠。

ps. 非常歡迎找我聊聊，不論什麼都可以聊，想更了解我或是合作都歡迎。

通常會看到這裡的應該都是 Lidemy 的朋友，先跟你說聲嗨，但不論是誰，都感謝你的閱讀。
