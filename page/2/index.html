TOEIC words

發表於 2021-02-18 分類於 英文筆記

Try not to use VERY

  • tire exhausted
  • tire => exhausted
  • hungry starving
  • hungry => starving
  • bad awful
  • bad => awful
  • good => excellent, awesome
  • angry => furious
  • passion => enthusiastic

Intensifier

really, so, especially, absolutely, quite, pretty, very, particularly
