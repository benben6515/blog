2023-04 English Interview

  1. 面試前的準備

    • 研究公司

      • 網路搜尋、新聞、社群
      • 產品、文化、客戶
      • 了解職位需求技能
      • LinkedIn

    • 了解面試流程

    • 準備問題和答案

      • 單向常見問題：自我介紹、為什麼想加入、為什麼離開現在的工作、可以為公司帶來什麼效應、為什麼覺得自己是適合的人選、對於未來發展的想法為何、期待能從工作中得到什麼
      • 回答技巧：準備清楚簡潔的答案、與個人經驗和公司價值觀相符合
  2. 常見的外商面試類型
    • Sourcing => Recruiter => Hiring Manager
    • Full cycle Recruiting
    1. Preparing
    2. Sourcing
    3. Screening
    4. Selecting
    5. Hiring
    6. Onboarding
    • Testing
      • Psychometric Aptitiude Test
      • Numerical Reasoning Test
      • Verbal Reasoning
      • Logical Reasoning Test
      • English Test
    • Behavior Interview
      • 介紹：互相介紹
      • 設定情境：如果跟上司不合、需要的即戰力
      • 問題提出
      • 追問疑點
      • 總結和結束
      • 應對技巧
      1. 避免重複敘述相同經驗，盡量真實回答
      2. 綀習以邏輯思考方式，STAR
      3. 提前練習行為面試，避免即興發揮和面試緊張狀況
      4. 在了解問題後，設定答案是短回履或是長回履
      • 短回覆可以是重點列出
      • 長回覆為了避免聽起了失去重點、長篇大論，可以說故事，盡量控制在 2 分鐘內
  3. 面試服儀及肢體動作
    • 表達能力
      • 用簡潔的語言達清楚觀點
      • 平穩的速度，看著自信地對方
    • 姿勢和肢體語言
      • 保持輕鬆，坐直
      • 帶筆跟筆記本
      • 面試前整理服裝儀容
      • 手機靜音
  4. 面試問題
    • 你的優缺點是什麼
    • 過去的職務中，最具有挑戰性或是最有成就感的是哪一個
    • 你如何解決困難或是衝突的情況？
    • 你對於工作時間和工作壓力有什麼看法？
  5. 薪資問題
    • 中高階人員，薪水浮動程度大
    • 拿到 Offer 後，可以談
    • 策略方式：自信堅定、認清現實、設定底價
    • 千萬不要：予取予求、不要獅子大開口

competitive offer

Q:

  1. If work years is short, but skills is quite well. How to convince hunter or recruiter you are qualified?
  2. It is believed that keep interviewing is good. It’s that true? If so, then how the frequency should be?
  3. I supposed most of Taiwan appliances are introverted. Is it good on the interview or not?
  4. Which is more important, accuracy of fluency, in the oral interview?

A:

  1. 舉例說明，你有能力做到，impact 是什麼，能為公司帶來什麼
  2. 是好的，可以取得
  3. 必需說內向是不好的，有的公司會議上不講話，公司就不需要你
  4. accuracy is better, how to represent list, maybe nervous,
